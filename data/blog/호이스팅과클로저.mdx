## 호이스팅

실행컨텍스트에 Record[^1]로 인하여 호이스팅이 발생하게 됨

```javascript
console.log(이름) // undefined

var 이름 = '이성규'

console.log(이름) // 이성규
```

이러한 현상이 발생하게 되는 이유는 실행컨텍스트를 공부하면 이유를 알 수 있음

- 실행컨텍스트를 구성 시 environment 에 식별자 정보 수집

- 이로 인하여 해당 컨텍스트 내부는 변수명들을 이미 암

**실행컨텍스트는 식별자를 관리 및 구분하는 용도** 이기 떄문
